# SkyCommerce Cloud-Native 🚀

A professional MERN-stack e-commerce platform built with a Cloud-Native approach.

## 🛠 Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js & Express
- **Database:** MongoDB
- **DevOps:** Docker, Docker Compose
- **Cloud:** Google Cloud Platform (GCP)

## 🐳 How to Run with Docker
```bash
docker-compose up --build
